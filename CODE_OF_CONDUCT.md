# 社区行为准则

> 像在跳海一样——真诚、友好、尊重彼此。

## 我们的承诺

我们致力于为每个人提供一个友好、安全、欢迎的环境，无论其经验水平、性别认同和表达、性取向、残障、外貌、体型、种族、民族、年龄、宗教或国籍。

## 我们的标准

**鼓励的行为：**

- 使用友好和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

**不可接受的行为：**

- 使用带有性暗示的语言或图像
- 人身攻击、侮辱或贬损性评论
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他在专业环境中不合理的行为

## 执行

如果发现违反行为准则的情况，请通过 Issue 或邮件联系项目维护者。所有投诉都将被审查和调查，并以适当的方式回应。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.1。
